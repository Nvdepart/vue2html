## Table of Contents


## Install

[npm][]:

```sh
npm install vue2html
```

[yarn][]:

```sh
yarn add vue2html
```


## Usage

```js
const Vue2html = require('vue2html');

const vue2Html = new Vue2html();

console.log(vue2Html.renderName());
// script
```
